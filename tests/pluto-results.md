# other abbreviations and shortcuts


### GET-SIZE

```
(GET-SIZE "interior-of-a-heart.txt")
```

Returns:
```
((GET-SIZE interior-of-a-heart.txt))
```


### GET-SIZE

```
(GET-SIZE "interior-of-a-heart.txt" :JUST-BYTES T)
```

Returns:
```
((GET-SIZE interior-of-a-heart.txt JUST-BYTES T))
```
