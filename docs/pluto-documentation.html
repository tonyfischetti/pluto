FOR-EACH/HASH:

hi

(LET ((TMP (MAKE-HASH-TABLE)))
  (SETF (GETHASH 'GREEN TMP) "veridian")
  (SETF (GETHASH 'RED TMP) "cadmium")
  (FOR-EACH TMP
    (FORMAT T "~A -> ~A~%" KEY! VALUE!)))

Output:

GREEN -> veridian
RED -> cadmium

FOR-EACH/LINE:

hi

(FOR-EACH/LINE "somebody.txt"
  (FORMAT T "~A -> ~A~%" INDEX! VALUE!))

Output:

1 -> we gotta celebrate diversity
2 -> in the university

FOR-EACH/LIST:

hi
